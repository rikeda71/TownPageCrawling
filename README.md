# TownPageCrowling

## やること
タウンぺージからの情報抽出

## 使い方
1. crawling.pyの75行目に指定されているカテゴリを変更する

    # デフォルトはsweetsカテゴリ
    category = "sweets"

2. 以下を実行

    python crawling.py # 時間めっちゃかかります

## その他
* ここを読んで使う[https://itp.ne.jp/guide/web/notice/index.html](https://itp.ne.jp/guide/web/notice/index.html)
* crawling.py のUserAgentは変更可
